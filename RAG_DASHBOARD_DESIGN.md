## Dashboard Architecture

### Main Components
1. **Navigation Sidebar** - Multi-page navigation
2. **Query Interface** - Natural language query input
3. **Results Display** - Retrieved code and context
4. **Analytics Panel** - Performance metrics and usage stats
5. **Configuration Panel** - System settings and API management
6. **Repository Monitor** - Repository indexing status
